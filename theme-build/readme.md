# Peanut Theme Build System

This build system generates css and js files for a peanut theme.  It was adapted from the Basic Bedrock theme build system. 

You can get the original source and installation instructions from
[The GitHub repository 'afixia-theme-basic-bedrock'](https://github.com/davedew/afixia-theme-basic-bedrock).

Or you can download the Basic Bedrock theme from [Concrete CMS Marketplace](https://www.concretecms.com/marketplace/themes/basic-bedrock).

To compile, open a command prompt in this directory and run `npm run dev` or `npm run prod`. The 'prod' script will minify the CSS and JS files for production use.

Output files will be generated in the `theme-build/output` directory.
