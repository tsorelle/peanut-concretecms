Installing a Markdown Extension

Browser Extensions are available for Edge, Chrome and Firefox

  • Install for Chrome
  • Install for Edge

Once installed, you may need to change some settings to make it work.

  1. Click the markup icon
  2. Click the "Advanced Options" button
  3. Select the "CONTENT" tab
  4. Under "Site Access" enter our url or click "ALLOW ALL"
