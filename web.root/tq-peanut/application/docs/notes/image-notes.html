Image Notes

Thumbnails

An image that is used for a news feed article (blog) should have a "Blog Entry" thumbnail.

The ideal dimensions for a thumbnail are 660 x 428. In most cases a thumbnail will be generated but if not, or if the generated thumbnail you will want to edit the original image or in the Concrete CMS image editor.

Hero Images:

The ideal size for a full-screen hero image is 1,200 pixels wide with a 16:9 aspect ratio. For a banner hero image, the ideal size is 1600 x 500 pixels.
